Get it Back! – Roguelike Shooter

Unity prototype · Top–down action & base-building · Role: Game Designer

Overview

Get it Back! is a fast-paced top–down shooter where each run is about pushing your luck to recover stolen items that upgrade the player’s home. The core loop mixes tight combat with small, meaningful progression choices between runs.

Design focus

  • Readable combat with clear feedback and short time-to-fail.
  • Risk / reward decisions around how far to push each run.
  • Base upgrades that subtly change how players approach new runs.
