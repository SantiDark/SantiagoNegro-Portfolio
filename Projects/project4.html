Riftbound – Tower Defense Roguelike

Concept & systems · Run-based tower defense with base-building meta · Role: Game Designer

Overview

Riftbound mixes classic lane-based tower defense with roguelike structure. Each run sends the player through escalating waves, earning resources that can be invested into a persistent base that unlocks new towers, modifiers and strategies.

Design focus

  • Core loop connecting runs, rewards and base upgrades.
  • Modifiers that change how players think about each new run.
  • Difficulty curve tuned around learning, not just grind.
